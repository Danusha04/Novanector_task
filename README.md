# Password Strength Tester

A simple GUI application to test the strength of a password. The tool evaluates passwords based on length, character diversity, common password lists, dictionary words, and entropy.

## Features

- Check password strength based on multiple criteria.
- Provide feedback on password weaknesses.
- Simple and intuitive GUI using Tkinter.

Install the required dependencies:
    ```bash
    pip install tkinter
    ```

## Usage

1. Run the script with Python:
    ```bash
    python password_strength_tester.py
    ```

2. Enter a password in the input field and click "Evaluate Password" to check its strength.
